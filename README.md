# 🌱 Campus Green Guardian - 校园绿色守护计划

## About This Project

An interactive, gamified environmental awareness application designed to encourage campus community members to participate in eco-friendly activities.

## Features

✨ **Interactive Tree Growth** - Watch your tree grow with every environmental action
- Complete eco-tasks (clean plate, water conservation, recycling)
- Click the button to add leaves to the community tree
- Track total participation count

🎨 **Beautiful Design** - Clean, modern interface with smooth animations
- Responsive mobile-friendly layout
- Satisfying leaf growth animations
- Green eco-friendly color scheme

🌍 **Community Impact** - See the collective environmental effort
- Real-time counter of participants
- Visual representation of community impact

## How to Use

1. Visit the [live website](https://g-ipgp21301074.github.io/campus-green-guardian/)
2. Complete an environmental task (节能、节水、环保)
3. Click the button: "我今天保护地球了！"
4. Watch your leaf appear on the tree
5. Share your impact with friends!
